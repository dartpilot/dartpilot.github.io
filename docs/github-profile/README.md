### Flutter Enterprise Starter

Production-ready Flutter template with clean architecture, BLoC pattern, and 6-platform support.

- Feature-first clean architecture with BLoC state management
- Auth & role-based access control out of the box
- 14+ reusable components with responsive design system
- Internationalization (EN/TR) with mock mode for rapid development
- CI/CD pipeline with GitHub Actions
- 472+ tests

```bash
git clone https://github.com/dartpilot/flutter-enterprise-starter.git
cd flutter-enterprise-starter
flutter pub get && flutter run
```
