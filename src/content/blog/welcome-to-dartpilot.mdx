We're excited to introduce **dartpilot** — a community-focused organization dedicated to building production-ready, open-source templates and tools for the Flutter/Dart ecosystem.

## Why dartpilot?

Building a Flutter application from scratch with proper architecture, state management, testing, and CI/CD can take weeks. dartpilot templates give you a head start with battle-tested patterns and best practices.

## Our Flagship Project

**Flutter Enterprise Starter** is our first template, featuring:

- **Feature-First Clean Architecture** — modular, scalable project structure
- **BLoC State Management** — with use cases and equatable states
- **6-Platform Support** — Android, iOS, Web, macOS, Linux, Windows
- **472+ Tests** — comprehensive test coverage
- **14+ Design System Components** — responsive and accessible
- **Mock Mode** — develop without a backend
- **CI/CD Pipeline** — GitHub Actions for build, test, and deployment

## Get Started

```bash
git clone https://github.com/dartpilot/flutter-enterprise-starter.git
cd flutter-enterprise-starter && fvm use
fvm flutter run --target lib/main/main_local.dart
```
